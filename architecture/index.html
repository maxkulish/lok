What This Project Does

Lokomotiv (lok) is a local CLI tool that orchestrates queries across multiple LLM backends (Claude, Codex, Gemini, Ollama, AWS Bedrock). It provides parallel querying, multi-round debates between models, workflow pipelines, and built-in tasks for code analysis, bug hunting, security audits, and CI failure diagnosis. All processing is local - it only makes network calls to the LLM APIs you configure. No data is sent to any third-party analytics or telemetry services.
